etvec
===============================

version number: 0.0.1
author: Sigrid Klerke

Overview
--------

Extract gaze vectors from raw gaze recordings

Installation / Usage
--------------------

To install use pip:

    $ pip install etvec


Or clone the repo:

    $ git clone https://github.com/SigridK/etvec.git
    $ python setup.py install
